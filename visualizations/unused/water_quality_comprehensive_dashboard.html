Contamination Flow Animation

What This Analysis Shows

This animated visualization tracks the spatiotemporal migration of arsenic contamination through the Pilcomayo River water system from 2016-2024. Each station is represented as a bubble whose size and color indicate contamination severity.

Key Insights
